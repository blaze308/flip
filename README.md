# AncientFlip

AncientFlip is a modern social platform where you can share moments (stories + posts), connect in real-time via chat, and jump into live video/audio experiences.

## What you can do

- Create content: stories and posts in multiple formats (text, images, video, and audio)
- Explore feeds: a home feed with reels-style video viewing and an immersive viewer
- Chat instantly: real-time messaging with typing indicators, reactions, replies, and media messages
- Express with gifts/animations: send Lottie/SVGA animations and gifts inside chat
- Go live and call: live streams and audio/video parties/calls (powered by Agora + Zego)
- Go premium: VIP/MVP/Guardian memberships with in-app purchases and subscription history
- Earn & manage coins: wallet/coin features backed by the app’s in-app purchase system
- Join the creator ecosystem: an agency system for hosts/agents

## Tech stack

- Flutter (Dart) + Riverpod (state management)
- Firebase (Auth + core services)
- Socket.IO (real-time chat)
- Agora RTC + Zego (live video/audio & calls)
- In-app purchases (Android/iOS)

## Setup (local development)

1. Open this folder:
   - `cd flip`
2. Update environment variables:
   - Edit `flip/.env` (required keys include `ZEGO_APP_ID`, `ZEGO_APP_SIGN`, `AGORA_APP_ID`, and `API_URL`)
3. Install dependencies and run:
   - `flutter pub get`
   - `flutter run`

## Notes

- Some features depend on third-party credentials being present in `.env`.
- Firebase is initialized using `firebase_options.dart` (generated by `flutterfire`).
